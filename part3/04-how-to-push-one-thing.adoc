== 成事心法：如何推动一件事？

一个小孩看上了一款心仪的玩具，如果家里是妈妈管钱，那么他应该去找妈妈要，而不是找爸爸。即使爸爸同意购买了这个玩具，他也不一定能成功。

这就是我们在推动一件事落地过程中经常遇到的问题，如何同干系人打交道。

本节我们讨论如何获得他人的支持。

=== 干系人分析：找到真正的决策者

推动事情的关键是找到真正的决策者。就像小孩要玩具需要找管钱的妈妈一样，工作中我们需要识别谁有真正的决策权和影响力。

干系人分析从两个维度来看：

- 影响力：对目标期望的促进作用，影响力大的人能决定成败。
- 动机：干系人和既定目标的利益诉求是否同步，利益一致才能获得极大支持。

干系人地图和 RACI 模型是常用的分析工具。

在干系人地图中，我们可以通过权力和利益两个维度，把干系人分为几类：

- 权力高，利益相关大：这是我们的关键干系人，需要重点管理。
- 权力高，利益相关不大：一般不需要让他们推动，但是不要得罪，避免他们反对即可。
- 权力低，利益相关性大：保持联系，但不需要过多在乎他们的意见，影响你的判断。
- 权力低，利益相关性小：关注他们的领导即可，不需要太浪费精力。

image::p3-stackholder.png[干系人地图]

关键是要找到那些在"重点管理"象限中的人——影响力大且利益诉求一致的关键干系人，找对人几乎是办事成功中最核心的问题。

除了干系人地图之外，RACI 模型是一个项目管理专业工具，可以把干系人地图中的关键干系人，分配到不同的角色中。

RACI 是 Responsibility（负责）、Accountability（问责）、Consulted（咨询）、Informed（知情）四个英文单词的首字母缩写，用来在项目中给每一位干系人分配清晰的角色，避免“多人负责、无人负责”或“信息黑洞”。

- R（Responsible）负责执行：具体干活的人，一个任务可以有多个 R，但不宜过多，否则容易扯皮。
- A（Accountable）最终负责：任务成败的唯一“背锅人”，只能有一个 A，权责对等，避免多头领导。
- C（Consulted）需被咨询：双向沟通，任务开始前或关键节点必须征求意见，通常是领域专家或上下游接口人。
- I（Informed）需被知会：单向告知，任务进展或结果发生变化时及时同步，保证信息透明，减少“我以为你知道”的意外。

用法示例：  

把干系人列在横轴，任务列在纵轴，在交叉格子里填 R/A/C/I。一张表就能让所有人一眼看懂“这事我该找谁、谁找谁、谁拍板”。  把干系人地图里“权力高、利益大”的关键人优先放进 A 或 R，再为“权力高、利益小”的人安排 C 或 I，就能既保证决策效率，又不遗漏任何潜在阻力。

为了影响这些干系人，就需要下面这些具体的成事手段了。

=== 核心原理：威逼利诱

这里的“威逼利诱”只是一个比喻，意思是推动事情落地，往往只有两种途径：

- 威逼：让对方觉得害怕，不得不这样做。害怕包括通过行政权力命令，亦或者通过让其害怕失去某些东西。
- 利诱：让对方觉得有好处，愿意主动去做。利诱包括通过奖励、认可、承诺等方式，来激发对方的行动，甚至卖个人情也算利诱，人情毕竟在以后可能要还的。

在上面小孩买玩具的例子中，找妈妈要钱这算是一种利诱。

父母通过抚养小孩的成就感获得奖励，这是父母愿意为小孩买东西的本质原因。如果在这个过程中父母，无法获得足够的情绪价值，可能就不愿意为小孩购买玩具了，所以情商高、嘴甜的小朋友更容易获得想要的玩具。

这是建立在未成年人推动事情落地的手段有限的前提之上的。在工作中，权力越大，威逼利诱的手段就越多，反之乖乖听话就行。

作为管理者，假设需要推动一个新的技术栈落地，可以：

- 通过激励措施，使用新的技术栈的团队成员进行奖励。【利诱】
- 如果某个技术能提高效率，通过公开数据看板，让不采用这个技术的人感受到压力。【威逼】
- 设定最终考核时间，如果不通过新技术学习的人，会进入绩效不合格的状态。【威逼】
- 经过一段时间，大部分人都能使用新技术，只有个别人还不会使用新技术，这些人会感受到危机。【威逼】
- ……

威逼和利诱在大的经济环境中也能发挥作用。比如房价上涨时，如果你不及时买房，在后面需要买房的时候就无法负担其价格了，这也是一种威逼；同时，房价上涨带来的溢价，这也是一种利诱。

在这两种机制的作用下，人们就疯狂的涌入房地产市场了。

=== 趋利避害：制造行动的条件

人行动的根本原因是趋利避害。不能因为"我想"别人就会行动，而是要制造令人行动的条件。

工作无法推动的原因通常有：

1. 相关人员没有切身利益，所以不支持也不拒绝。
2. 相关人员利益受损，强烈反对。
3. 有利益驱动的人员没有站到关键位置上，无法抵消利益受损人员的影响。

所以我们需要把自己的诉求，包装成他人的利益或者价值观。这一点在营销领域经常使用。

为了卖出东西，他们可能会包装成：

- 爱国：某汽车品牌。
- 投资：把房产包装成资产。
- 公益：助农电商直播，以公益之名，实际上卖自己的东西。
- 爱情：钻石骗局，把一颗石头和爱情绑定，让人付出高溢价。

=== 核心团队：建立你的制胜联盟

如果你想推动一大群人，可以先推动一小群人。比如，很多商家为了营造出非常抢手的效果，经常早上通过免费赠送东西雇人排队。

这一小部分人叫做制胜联盟，我们在前面关于组织部分讨论过这个问题，因此也可以用它来推动事情落地作为实践。

核心团队（Core Team）是管理者一项非常有力的工具，就像充满水雾的积雨云需要等待较大颗粒的灰尘作为凝结核一样，Core Team可以作为引擎，去启动更大的团队。

这小撮人能够：

- 驱动更多人行动
- 塑造团队文化  
- 建立团队秩序
- 传达任务
- 协调资源

核心团队就像是一个托。如果只是你自己摇旗呐喊，被影响的人很少，但是如果你先鼓动2-3个人支持你，然后一起行动，其他人就也会被影响了。

当然，核心团队不宜过大，否则会导致利益难以保持一致，意见无法收敛，认同感减弱。当被影响的人越来越多，就会慢慢形成趋势。就像在蝴蝶效应中，在美国西海岸的天气会影响到整个美国的天气，这就是趋势在形成。

而如果本来就有宏观趋势呢？那就更好了，一旦识别到趋势，我们也要及时利用它。

=== 借势造势：利用趋势的力量

当我们想要推动一项事情时，如果不具备宏观的趋势，会"诸事不顺"。信息技术行业的精英并非比其他行业的人强多少，但由于信息技术是时代的趋势，没有人能阻挡，推动起来就相对容易。

这种趋势的效益不仅发生在宏观上，日常生活工作无处不在。有时候趋势很微妙，一个人提出一件对公司有益的事，但是响应者寥寥，换个人就完全不同。这是每个人身上带有的影响力不同造成的。

推动策略包括：

- 上升：如果干系人配合度不高，就从其他地方借力，如果他的领导能认同既定目标，这样对于干系人的 KPI 就有了保证。
- 造势：如果大家都往前走，而某些干系人不动，其实就是后退。找到办法制造既定目标的趋势，这样对于其他干系人就会产生危机感。

要想上升和造势，就必须抱一个大腿，借助其他人的力量。

在《让子弹飞》中有一个桥段是县长和黄老爷商量出城缴匪，然后让豪绅带头捐款，然后让百姓捐款，事成之后，豪绅的钱如数退还，百姓的钱和豪绅 3-7 分成，7成还是人家的。

这个桥段虽然颇为讽刺，但是很多时候却真实反应了现实的运作模式。如果自己势单力薄，比较好的方式就是抱上一个平台的大腿，利用平台的能量和趋势，来推动自己的事情，虽然收益少一点，但成功的概率会大大增加。

比如如果一名技术讲师，想要发布自己的技术课程，在自己没有流量的前提下，想要赚到钱非常困难。而如果有课程平台原因合作上架课程，虽然会分走一部分收益，但是反而能获得更多的最终收益。

在职场内部其实也是一样。如果你想推动什么，没有高层作为支持就会困难重重，如果能把这件事的收益作为高层的绩效，那么就会很容易成功，即使做成之后，大部分的绩效都被人拿走了，也没必要眼红，双赢更重要。

=== 沟通的艺术：用叙事来包装你的目的

没有人喜欢被强加的观点，我们喜欢的，是自己得出结论。所以故事的引导作用就显现出来了。

通过讲故事、案例、比喻、引用等方法来说明问题更容易让人接受。永远不要试图给别人强加一个观点，除非他为此付了钱。演讲是实现影响力的重要方法。演讲主体不是演讲者"我们"，而是听众"他"。我们要深谙"他"的"听讲逻辑"，而不是"我们"的"演讲逻辑"。

而演讲中，最重要的就是叙事，也就是 Storyline。无论汇报、销售，最重要的就是叙事情节合理发展，最终你的目标让听众合理化，最终为你的提案买单，这是一项超强技术的修炼。

《故事经济学》中讲了人类是如何通过故事来影响经济的发展，传统经济学模型往往忽略了叙事，仿佛人类是完全理性、不受故事影响的个体。

实际上人类很容易被故事情节吸引，不同的叙事会带来不同、甚至截然相反的结果。就像在战国时期，纵横家用不同的叙事方式，直接让国君选择对抗还是合作。

无论是大萧条、互联网泡沫还是次贷危机，其背后都有一套强大的、被广泛接受的“故事”在推波助澜。一个故事（如“房价永远上涨”）会影响人们行为，行为反过来强化叙事和价格，形成自循环，最终可能导致泡沫或恐慌。

比特币是一个绝佳的叙事经济学案例。它的价值几乎完全建立在一种强大的、技术乌托邦式的叙事之上。同样的，在其他环境下，想要推动一件事情落地就需要一个好的叙事。

在企业管理中，CTO 想要推动更先进的信息化系统，所以业界就有了”数字化转型“的叙事，当领导层认同了这个叙事，就会有源源不断的资金和支持。

即使在家庭中，孩子妈妈想要争取家庭的支持，让孩子去更好的学校，就会用教育即未来这个叙事，即便是更好的学习只是教育的一部分，甚至不是关键的要素。

如何通过叙事让他人接受你的观点，这是我们下一节进一步讨论的问题。
