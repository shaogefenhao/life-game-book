= life-game-book

《人生游戏：自我觉醒、社会规则和执行力》一本关于被社会反复毒打后认清世界真相的书，注定无法出版，遂开源，希望对有缘的人看到有所帮助。

下面是前言部分：

“朝闻道，夕死可矣”。我们一生都在追求尝试理解这个世界，回答三个终极问题：我是谁、我从哪里来、我到哪里去？以及根据我们的理解，采取更好的策略在世界中生存。

我有幸读了很多书，对这几个问题有了一些见解，并拓展出整个《人生游戏》的框架，希望在有限的生命中，能把它们分享出来，写了三个专栏，最终集结成书。

这三个专栏以问题驱动，每个专栏包含了 10 个问题，分别是：

- 《人生十问：自我认知的困难问题》。从进化的角度理解人性，讨论生存和繁衍、动机、欲望和意识。让你找回主体性，摆脱抑郁和焦虑。
- 《人生十问：社会的游戏规则》。从系统的角度分析组织，讨论合作、竞争、博弈和群体认同。让你知道这个社会背后的逻辑规则，如何在这个规则下生存和繁衍。
- 《人生十问：成事，行动和执行力》。从执行力的角度，讨论行动、决策和改变自己的处境。

因此，这三个专栏构成了这本书的三个组成部分，分别对应了人生的三个阶段：自我认知、社会游戏和成事。

这本书不是鸡汤，也不是箴言集。它尝试用一套可检验的模型与规律，去解释人性、社会规则与执行力，并把理解落到可操作的行动上。

我们从生物与信息处理的底层机制出发，讨论意识、痛苦与幸福的本质：为何快乐转瞬即逝，羞耻与欲望为何成为行为的驱动力；为何“爱”常常是联盟的选择；为何操控如此常见、语言为何能改变决策，并如何用逻辑与清醒抵御“语言注入”。我们拒绝把观点当作真理，只把“有用的模型”当作工具。

当视角从个体切换到群体，社会不再是抽象的叙事，而是一个有约束的系统：博弈、权力、道德与宗教都被还原为资源分配与秩序维护的机制。我们讨论组织如何运转、动机如何发生、权力如何形成与制衡，可以帮助我们理解社会的运行原理，从而更好地应对社会的挑战。

最后，所有理解都要落在行动上。如何深度分析问题、在不确定中做出更好的决策、如何把事推到落地、如何影响他人与建立关系、如何保持专注与节奏、如何在挫折中复原，直至形成稳定的内在核心。方法论不是精致的流程图，而是能在复杂环境中反复奏效的实践手册。

写作原则是理性、克制与可操作：尽量用清晰的语言、低噪的结构，把复杂问题拆解为可执行的步骤。愿它成为一本“能用”的书，帮助你在真实世界里做成事情，同时更清晰地看见自己与世界。

== 如何为仓库贡献内容

选择一个 Asciidoctor 编辑器：

- IntelliJ + Asciidoctor 插件
- Visual Studio Code
- AsciidoctorFx

按章节编写内容。

== 如何构建电子书

Mac 先安装 Asciidoctor：

 $ brew install asciidoctor

生成 HTML 文件：

 $ asciidoctor -vwt -o docs/index.html index.adoc

生成 PDF 文件：

 $ asciidoctor-pdf -o docs/life-game-book.pdf index.adoc

生成 Epub3 文件：

 $ asciidoctor-epub3 -o docs/life-game-book.epub index.adoc
